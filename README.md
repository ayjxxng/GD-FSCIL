# DG-FSCIL

## Requirements
Please see `pyproject.toml` or using `uv` to sync requirements.

## Datasets
To download raw image data, run `data/download_dataset.py`.

## Running scripts
`python main.py` or `uv run main.py` if you are using `uv` venv.
